# Codigo CRUD para usuarios
Este programa esta diseñado para dar altas a usuarios, login, mostrar los demas usuarios, actualizar usuario y eliminar usuario.  
**Este codigo cuenta con las siguientes caraccteristicas:**
- Arquitectura  
	- Uso de arquitectura Modelo, Vista y Controlador para la comunicacion entre el usuario y el servidor.  
	- Uso de inyeccion de dependencias, para comunicación interna entre los componentes.  
	- Uso de ROP (Rail Road Oriented Program) para el flujo de los datos, comunicacion con la base de datos y manejo de excepciones.  
	- Creacion de la base de datos enfocado a CodeFirst.  
- Tecnologias de backend  
	- Uso de entityFramework como ORM para el modelado de la base de datos.
	- Uso de Data annotation para control de de exepciones.
	- Uso de la libreria ROP por netMentor, para el flujo de los datos y aplicar ROP en el proyecto.
- Tecnologias de fronend
	- Uso de Bootstrap para el uso de plantillas de estilo.
	- HTML Nativo para la maquetacion del proyecto.
	- CSS Nativo para los estilos del proyecto.
